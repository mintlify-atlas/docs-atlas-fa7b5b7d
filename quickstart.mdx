---
title: "Quickstart"
description: "Run your first Brasileirão match prediction in under 5 minutes."
---

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/mtgr18977/sassamaru-br-25.git
    cd sassamaru-br-25
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pip install pandas
    ```

    For the desktop GUI, also install `numpy` — tkinter is included with most Python installations:

    ```bash
    pip install numpy
    ```
  </Step>

  <Step title="Prepare your match data">
    The file `br-25.csv` is already included in the repository under `sassamaru-br-25/br-25.csv`. No download required.

    The CSV must have four columns:

    | Column           | Description                  |
    | ---------------- | ---------------------------- |
    | `mandante`       | Home team name               |
    | `visitante`      | Away team name               |
    | `gols_mandante`  | Goals scored by the home team |
    | `gols_visitante` | Goals scored by the away team |

    Example rows:

    ```csv
    mandante,visitante,gols_mandante,gols_visitante
    flamengo,palmeiras,2,1
    corinthians,sao paulo,0,0
    atletico mineiro,gremio,3,1
    internacional,botafogo,1,2
    ```

    <Note>
      Team names are normalized to lowercase internally — `df['mandante'].str.strip().str.lower()` — so casing in the CSV does not matter, but lowercase is recommended for consistency.
    </Note>
  </Step>

  <Step title="Configure your fixtures">
    Open `previsao.py` and edit the `jogos` list inside `main()` to set the matches you want to predict. Each entry is a tuple of `(mandante, visitante)` — home team first, away team second — written in lowercase:

    ```python
    jogos = [
        ('internacional', 'vitoria'),
        ('bahia', 'atletico mineiro'),
        ('corinthians', 'bragantino'),
        ('cruzeiro', 'gremio'),
        ('fortaleza', 'ceara'),
        ('juventude', 'sport'),
        ('flamengo', 'sao paulo'),
        ('vasco', 'botafogo'),
        ('santos', 'palmeiras'),
        ('mirassol', 'fluminense'),
    ]
    ```

    Add, remove, or reorder rows freely. Every team name must match a team present in `br-25.csv` so the model has historical data to draw from.
  </Step>

  <Step title="Run the prediction">
    ```bash
    python previsao.py
    ```

    The model reads `br-25.csv`, builds Poisson strength ratings and Elo histories from historical results, then prints a Markdown table to stdout:

    ```markdown
    # Previsão de Jogos - Modelo Híbrido Poisson + Elo

    | Mandante      | Visitante         | Gols Mandante | Gols Visitante | Prob Mandante (%) | Prob Empate (%) | Prob Visitante (%) | Palpite   |
    |:------------- |:----------------- | ------------: | -------------: | ----------------: | --------------: | -----------------:|:----------|
    | Internacional | Vitoria           |          1.23 |           0.98 |              42.1 |            29.5 |               28.4 | Mandante  |
    | Bahia         | Atletico Mineiro  |          1.10 |           1.05 |              35.0 |            33.0 |               32.0 | Empate    |
    ```

    **Output columns explained:**

    | Column              | Description                                                                 |
    | ------------------- | --------------------------------------------------------------------------- |
    | `Mandante`          | Home team                                                                   |
    | `Visitante`         | Away team                                                                   |
    | `Gols Mandante`     | Expected goals for the home team (Poisson λ adjusted by Elo)               |
    | `Gols Visitante`    | Expected goals for the away team (Poisson λ adjusted by Elo)               |
    | `Prob Mandante (%)` | Probability the home team wins, summed across all scorelines                |
    | `Prob Empate (%)`   | Probability of a draw                                                       |
    | `Prob Visitante (%)` | Probability the away team wins                                             |
    | `Palpite`           | Predicted outcome — the outcome with the highest probability               |
  </Step>
</Steps>

<Note>
  Prefer a graphical interface? Run `python sassamaru.py` to open the desktop GUI, which provides the same predictions in a point-and-click window. Requires `numpy` in addition to `pandas`.
</Note>
